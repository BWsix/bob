# Bob - 學習歷程檔案系統

### 值得信賴

Bob 用心保管確保您的檔案，確保不會有無故遭刪除的風險

### 盡情發揮

Bob 將敘述的 100 字上限提升到 420 字，讓文思泉湧的您也能盡情發揮

### 減少失誤

Bob 支援檔案預覽，讓您不怕選錯附加檔案
